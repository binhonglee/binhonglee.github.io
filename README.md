## Build Setup

``` bash
# install dependencies
pnpm install

# serve with hot reload at localhost:8080
pnpm run serve

# build for production with minification
pnpm run build

# run unit tests
pnpm run test:unit

# run e2e tests
pnpm run test:e2e
```
